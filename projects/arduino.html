Embedded Systems

An embedded system is a computer system with function within a larger system to perform a specific task. It usually consists a combination of components including a processor, memory, input/output devices. For this section of embedded systems, I will be using TinkerCAD to create the circuits for the embedded systems.

Programming Assignments

Simple LED Control


Written program to blink the LED.

Controlling 3 or more LEDs


Written program producing running light in sequence of R -> G -> W -> so on.

Adding a switch to the flashing lights


Written program to have flashing lights turned on when push button is pressed, and LEDs turned off when push button is pressed again. However, one flaw in this program is that you need to hold the push button till the white LED turns off before it returns all the LEDs to the off state.

Switch to control type of light-up


Written program to have a push button to cycle through the type of light-up in the sequence of, OFF > RED > GREEN > WHI > Flashing > OFF..., repeatedly. Similar to the previous program, you need to hold the push button between transition of light.
