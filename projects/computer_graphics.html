Computer Graphics

Computer graphics consists of two catergories, raster and vector. Computer graphics softwares are used to digitally manipulate visual contents like images.

What is Raster Graphics

Raster graphics is mainly compiled of pixels which are displayed on screen and it is used for photographs and images which all consists of pixels. The file formats include, .bmp, .jpg and .png.

What is Vector Graphics

Vector, unlike raster graphics which is made up of pixels, is made up of lines, and as the image is zoomed in, it does not lose quality. The file formats include, .dxf, .svg and .STEP.
