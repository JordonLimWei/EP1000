Tools

At the beginning of this module, we were tasked to research and select software tools that we would like to use for digital fabrication and other applications. Below is a table to tools that I personally chosen with the corresponding reasons as to why I chose them.
Purpose Choice Reasons
Text Editor Atom As introduce by my lecturer, it is said that atom is similar to SublimeText3 which is his editor of choice. Selecting Atom will be easier to refer to.
Graphics Editor GIMP GIMP is a graphics editor rivalling Photoshop. Since Photoshop is a paid software, the free alterative would be GIMP.
Vector Editor InkScape InkScape have numerous videos online for me to watch, making it easier for me to learn this vector editing software.
Screen Capture PicPick Out of all the selection of screen capturing softwares, only PicPick is available for Windows operating computers which I am using.
Image Resizer FastStone Photo Resizer It is a very fast and easy to use software to convert photos to sizes that you want.
CAD System Fusion 360 Though I have learnt other software like AutoCAD and Inventor, I would love to learn another CAD software to my arsenal.
IDE Arduino Since we are using Arduino boards to do our electrical circuits in our project, this is a must learn.
Browser Chrome I have always been using Chrome ever since having troubles with accessing websites using browsers such as Internet Explorer.
